¿Qué es la sentencia if?

La sentencia if es una estructura de control en programación que permite tomar decisiones. Se utiliza para ejecutar un bloque de código solo si se cumple una condición.

Es decir, el programa evalúa si algo es verdadero o falso. Si la condición es verdadera, se ejecuta una acción; si no, puede ejecutar otra usando else.
